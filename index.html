Projects

statusbar

A primitive statusbar intended for dwm. It's really just a chain of text passed to xsetroot, grabbing info for battery and volume using upower and alsa, auto-updating every 60 seconds by calling itself again. I have set up my own dwm instance so that upon certain keystrokes (e.g. volume buttons) it calls the script in order to refresh it. It also displays the current keyboard map, and I had intended to show the state of auto-capitalization by capitalizing the map name when present but have not gotten around to doing so.

ttrpgbot

Basic bot used as a test of the Python Discord API. Intended to be used for virtual tabletop gaming sessions, so included commands to roll dice and join voice chat. Have not touched in a while, but had intended to look into playing audio from sources such as YouTube.

manualmonster

A work-in-progress attempt to create a webform storing input data as json and an interface to parse a collection of said json. Namesake is due to its intended purpose of organizing large amounts of data relating to monsters in a certain tabletop RPG.
